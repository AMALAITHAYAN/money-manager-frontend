# Money Manager (Frontend)

React + Tailwind frontend for Money Manager (Spring Boot + MongoDB).

## Run locally
1. Install deps:
   ```bash
   npm install
   ```
2. Start dev server:
   ```bash
   npm run dev
   ```
3. Ensure backend is running at:
   - http://localhost:8080

## Environment
`.env`:
- `VITE_API_BASE_URL=http://localhost:8080/api`
